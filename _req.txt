`n 6. Mock-up dell'Applicativo `n I mock-up seguenti rappresentano la visione progettuale dell'interfaccia dell'Albo Fornitori Digitale del Gruppo Solco. 24 schermate coprono l'intero flusso operativo. `n `n Le tavole sono in due sezioni: 6.1 Front-end (area pubblica e fornitore) e 6.2 Back-end (area amministrativa). Sono mock-up ad alta fedeltà, base per la fase di sviluppo UX/UI. `n `n 6.1 Front-end — Area Pubblica e Fornitore `n Le schermate mostrano l'esperienza del fornitore: landing page, wizard iscrizione Albo A e B, area riservata, conferma invio e template e-mail. `n `n ● FE Landing Page Pubblica — Selezione Albo A / Albo B `n La pagina pubblica di accesso presenta le due card di iscrizione con icone, descrizioni e vantaggi. Design con palette brand Solco. `n `n `n `n ● FE Wizard Iscrizione — Sezione 1: Dati Anagrafici `n Step-by-step con barra avanzamento a 5 fasi. Validazione inline con errore CAP evidenziato, badge salvataggio automatico, upload foto profilo. `n `n `n `n ● FE Wizard Iscrizione — Sezione 3A: Ambiti Tematici Docente `n Lista ambiti con checkbox: alla selezione si espandono campo specifica tematiche e selettore anni esperienza. Campo presentazione con contatore caratteri. `n `n `n `n ● FE Area Riservata Fornitore — Dashboard Personale `n Banner stato attivo con barra rinnovo. KPI card (punteggio, collaborazioni, documenti, completamento). Profilo pubblico, documenti con validità e comunicazioni. `n `n `n `n ● FE Wizard — Sezione 2: Selezione Tipologia Professionale `n Griglia 5x2 di card tipologia. Card Docente/Formatore selezionata evidenziata. Callout con implicazioni scelta. Campo ATECO per categoria Altro. `n `n `n `n ● FE Wizard — Sezione 5: Dichiarazioni e Consensi `n Tre blocchi: dichiarazioni legali, privacy GDPR, Codice Etico. Consensi facoltativi distinti. Firma OTP. Pulsante invio abilitato solo con tutte le voci obbligatorie. `n `n `n `n ● FE Wizard — Riepilogo Pre-Invio `n Tre card affiancate (Dati personali, Competenze, Disponibilità) con pulsante Modifica. Strip verde dichiarazioni. Pulsante invio verde prominente. `n `n `n `n ● FE Conferma Invio — Codice di Protocollo `n Cerchio verde checkmark, codice protocollo in evidenza con copia. Card: e-mail conferma, tempi revisione, iter integrazione. CTA area riservata. `n `n `n `n ● FE Wizard Albo B — Sezione 1: Dati Aziendali `n Campi societari (P.IVA, REA, CCIAA, data costituzione), sede, contatti, legale rappresentante e referente operativo per Gruppo Solco. `n `n `n `n ● FE Wizard Albo B — Sezione 3: Selezione Servizi `n Accordion cinque categorie. Per ogni categoria aperta: checkbox servizi, counter selezionati e campo descrizione obbligatorio. Banner riepilogo categorie attive. `n `n `n `n ● FE Template E-mail — Approvazione Iscrizione `n E-mail con header blu Solco, banda gialla brand, banner verde conferma, box dati iscrizione, tre punti azione e CTA area riservata. Rendering realistico client e-mail. `n `n `n `n `n 6.2 Back-end — Area Amministrativa `n Le schermate mostrano l'area admin: dashboard, lista fornitori, schede profilo, coda revisioni, valutazioni e report. `n `n ● BE Dashboard Amministrativa `n Home back-end: sidebar Solco, cinque KPI card, alert urgenze, coda candidature con countdown e grafico iscrizioni mensili Albo A vs B. `n `n `n `n ● BE Lista Fornitori Albo A con Filtri Avanzati `n Ricerca full-text e cinque filtri combinabili con badge rimovibili. Tabella avatar, tipologia, ambiti-tag, stelle e stato colorato. Paginazione numerata. `n `n `n `n ● BE Scheda Profilo Fornitore — Vista Amministratore `n Header banda stato, tab Profilo/Documenti/Valutazioni/Storico/Note/Comunicazioni. Pannello Note solo admin (giallo). Barra azioni fissa. `n `n `n `n ● BE Coda Candidature — Revisioni Pendenti `n KPI strip stati. Tab filtro. Badge urgenza rosso per oltre 5 giorni. Pulsante azione primario per urgenti, outline per le altre. `n `n `n `n ● BE Scheda Revisione Candidatura con Azioni `n Banner urgenza, header candidato, checklist documentale. Azioni rinnova/apri profilo per ogni stato. `n `n `n `n ● BE Form Nuovo Invito — Con Anteprima E-mail `n Layout due colonne: form (tipo albo, destinatario, durata, messaggio) e anteprima live e-mail con logo e link dinamico. `n `n `n `n ● BE Pagina Valutazioni Complessive `n Quattro KPI. Certificazioni ISO con scadenze. Servizi per categoria. `n `n `n `n ● BE Report e Statistiche `n Sei KPI, grafico iscrizioni mensili, ranking ambiti, distribuzione stati, top 5 per valutazione, esportazioni rapide Excel. `n `n `n `n
